Recent Projects

Conditional Logit Model

A conditional logit model can be used for modeling discrete choices of agents, that may have varying characteristics, between alternatives that have varying attributes.

Gibbs Sampling Algorithm

Given observed choices between three ranked alternatives, you can use Gibbs sampling with data augmentation to fit an ordered probit model. This Bayesian procedure generates a probability distribution for the model parameters.

Tailored Metropolis-Hastings Algorithm

The Metropolis-Hastings algorithm is a a Markov chain Monte Carlo method that simulates draws from a Bayesian posterior distribution. Given observed data and a proposed model, the method generates a probability distribution for model parameters.
